# 📝 Flutter To-Do App

# A modern and feature-rich To-Do application built with Flutter, designed using MVVM architecture for clean structure and scalability. The app uses Provider for efficient state management and follows best practices for maintainable mobile development.

# It includes task creation, editing, and deletion with Undo support, along with search, filtering, and due date management to improve productivity. The UI is built with Material 3, offering a smooth experience with full Dark Mode support.

# For data persistence, the app uses Hive for fast and lightweight local storage, ensuring an offline-first experience with high performance and reliability.
